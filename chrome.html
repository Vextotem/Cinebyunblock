Step-by-Step Setup Guide

1

Access Chrome Flags

Open Chrome and type the following in your address bar:

chrome://flags

This will take you to Chrome's experimental features configuration screen.

2

Enable ECH Features

Use the search function to find and enable these two features:

  • encrypted-client-hello
  • use-dns-https-svcb-alpn

Set both options to "Enabled" using the dropdown menus.

3

Configure Chrome Settings

Navigate to Chrome Settings ‚Üí Privacy and Security ‚Üí Security

4

Enable Secure DNS

In the Security section, activate "Use secure DNS" and choose either:

  • Cloudflare (Recommended for speed)
  • Google (Alternative option)
5

Restart Browser

Close Chrome completely and restart it. Done! You can now browse without restrictions!
